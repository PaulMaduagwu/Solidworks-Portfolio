## Overview
This project involved the design of an all-terrain electric mobility vehicle intended for safe operation in rocky, icy, and wet environments. The design emphasizes user safety, ergonomics, and accessibility.

## Key Features
- Ergonomic seat with integrated seatbelt system
- Adjustable armrests for user comfort
- Modular remote controller mount adaptable for left- or right-handed users
- Front headlights for improved visibility
- Integrated charging port
- Glove compartment with automatic task lighting
- User-centered design focused on safety and usability

## Tools Used
- SolidWorks (Parts, Assemblies)
- Mechanical design principles
- Ergonomic and safety-focused design methodology
